# Changelog

All notable changes to `laravel-dodopayments` will be documented in this file.

## [1.0.0] - 2025-11-06

### Added
- Initial release
- Checkout Sessions integration
- Payment Links (static and dynamic)
- Webhook support with signature verification
- Laravel events for webhooks
- Pre-built checkout, success, and cancel views
- Facade for easy access
- Auto-discovery support
- Comprehensive documentation
- Test mode and live mode support
- Multi-product cart support
- Customer pre-filling
- Metadata tracking
- Configurable routes and URLs
